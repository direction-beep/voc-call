🔧 Améliorations Techniques Appliquées

1. Typographie de Base

body { font-size: 16px; line-height: 1.7; -webkit-font-smoothing: antialiased; -moz-osx-font-smoothing: grayscale; }

2. Titres Responsives

h1 { font-size: clamp(2.5rem, 5vw, 4rem); letter-spacing: -0.03em; line-height: 1.1; }

3. Textes Longs

.text-content { max-width: 75ch; line-height: 1.8; text-align: justify; hyphens: auto; }

4. Paragraphes Optimisés

p { font-size: 1.125rem; line-height: 1.8; color: var(--color-gray); max-width: 65ch; }
